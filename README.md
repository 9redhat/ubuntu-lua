# ubuntu-lua - Conky clock rings for Ubuntu
## Install
```
mkdir -p "$HOME/.config/conky"
cd "$HOME/.config/conky"
git clone https://github.com/tsbarnes/ubuntu-lua.git
```

## Usage
```
conky -c "$HOME/.config/conky/ubuntu-lua/conkyrc"
```
